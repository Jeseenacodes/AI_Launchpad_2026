# Founder Command Center Agent

## Overview

Founder Command Center is an AI-powered agent designed to help founders organize projects, identify priorities, track progress, and make better decisions. It acts as a daily operating system that transforms scattered information into actionable insights.

## Problem

Founders often manage multiple projects, tasks, ideas, and goals simultaneously. As information grows, it becomes difficult to identify priorities and maintain focus.

## Solution

The Founder Command Center Agent collects project updates, analyzes progress, identifies blockers, and generates recommendations for next actions.

## Features

* Daily Founder Brief
* Project Status Tracking
* Priority Recommendations
* Risk & Blocker Identification
* Weekly Goal Planning
* Actionable Next Steps

## Workflow

Input → Founder Command Center Agent → Founder Brief → Founder
