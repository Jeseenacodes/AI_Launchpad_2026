## AI Automation Projects

This repository showcases a collection of AI-powered automation workflows built using no-code and low-code platforms such as Zapier, Make.com, Relay.app, Google Workspace, and AI tools. These projects demonstrate how automation can streamline educational content workflows, technical interview preparation, marketing operations, and productivity while delivering practical, real-world solutions.

### Featured Projects

- **AI-Powered SQL Interview Question Generator** – Automatically generates customized SQL interview questions based on role, topic, and difficulty, helping learners prepare more efficiently.

- **Lesson QA Automation** – Uses AI to review lesson drafts for my educational apps (Little Noories, Tamilingo, and Little Readers), checking age-level suitability, clarity, vocabulary, quiz alignment, and formatting consistency.

- **AI-Powered LinkedIn Content Repurposer Automation** – Transforms a single project update into multiple content formats, including LinkedIn posts, cohort updates, and GitHub documentation, reducing manual content creation.

- **Parent Progress Summary Automation** – Automatically generates personalized learning progress summaries for parents using student activity data, improving communication and engagement across my educational apps.

- **Tamilingo AI Marketing System** – An AI-driven marketing workflow that streamlines content ideation, social media copy, visual asset creation, and campaign publishing to support consistent marketing for the Tamilingo language learning app.

Together, these projects reflect my interest in combining AI, automation, analytics, and product thinking to build scalable solutions that improve educational experiences, content operations, and business workflows.
